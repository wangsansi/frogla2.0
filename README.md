# 物业管理微信小程序

物业管理工单系统项目的小程序端，包括创建住户、创建工单、查看工单列表、查看工单详情等。

PC端：https://github.com/wangsansi/react-ticket-app

服务端：https://github.com/wangsansi/node-ticket-app

微信小程序+Tdesign组件库
